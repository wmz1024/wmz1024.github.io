教你如何用git和github-上传

对于github，git 开发者们一定很熟悉了
就是可以把你的代码上传到github上托管
灰常好用

安装

安装个git
这个我在以前已经发了→_→
往下翻就好了
也可以点这个链接
注册个github账号[贼简单 网上也有教程 我已经注册好了 一般打开github就让你注册>]

简单的上传

  • 如果你有一个仓库
    首先 在根目录下运行git clone [url]
    例如
    [url]可以代表 https:// github.com/用户名/仓库.git

    也可以代表git@github balabal
随便找一个克隆
之后cd到仓库名
例如
![2](https://gitee.com/wmz1024/img/raw/master/img/1601372806308.jpg)
里面就是内容了
修改内容就可以了
比如新建个文件
![3](https://gitee.com/wmz1024/img/raw/master/img/1601373381520.jpg)
[这个仓库必须是你自己的 不然上传可能就上传到别人仓库uh 这有点不可能 你不知道别人仓库的密码 用户名]
之后 让git认识你 这个我有点忘了 去百度搜搜吧
之后就是重头戏 上传
输入

    效果

    • 在本地新建仓库并上传到github
      新建一个文件夹
      cd到那个文件夹
      git init
      初始化这个仓库
      进入后 输入git remote add origin [url]
      [url]表示仓库地址.git
      新建文件
      上传 git add . && git commit -m "wmz blog" && git push -u origin master
