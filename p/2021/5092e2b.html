typora收费了？？？

typora收费是必不可免的事情 开发者开发出好用的工具 不收费怎么行呢 一开始typora没有收费 现在收费了 老用户必然会走 找别的去 此教程在一段时间之内typora还算好用 但时间推移 我们就可能会找其他的markdown编辑器

收费解决办法

我们首先找到他的老版本 目前正式版没有老版本 我们需要切换到dev

找到

Windows 64： https://download.typora.io/windows/typora-update-x64-1117.exe

Windows 32： https://download.typora.io/windows/typora-update-ia32-1117.exe

Linux 64： https://download.typora.io/linux/typora_0.11.18_amd64.deb

其他版本可以在这里找到

Typora for windows — beta version release
