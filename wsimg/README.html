Install instructions

To install this package:

Extract this package in <web site>/wsimg/. If your site is http://www.example.com, you should be able to access a file named http://www.example.com/wsimg/favicon.ico.

Insert the following code in the head section of your pages:

<link rel="apple-touch-icon" sizes="72x72" href="/wsimg/apple-touch-icon.png">
<link rel="icon" type="image/png" sizes="32x32" href="/wsimg/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/wsimg/favicon-16x16.png">
<link rel="manifest" href="/wsimg/site.webmanifest">
<link rel="mask-icon" href="/wsimg/safari-pinned-tab.svg" color="#5bbad5">
<link rel="shortcut icon" href="/wsimg/favicon.ico">
<meta name="apple-mobile-web-app-title" content="Wlog">
<meta name="application-name" content="Wlog">
<meta name="msapplication-TileColor" content="#603cba">
<meta name="msapplication-config" content="/wsimg/browserconfig.xml">
<meta name="theme-color" content="#ffffff">
